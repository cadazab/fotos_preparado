# foto preparau

## to work:
 * go to directory
 * `npm install`
 * `npm run dev`
 * go to link that is displayed
 
## to upload:
 * `npm run build`
 * open `dist/index.html` and add background color to the body (e.g. `background-color: red;`)
 * content of newly created subdirectory `dist` to web server
